# Selenium WebDriver Learning (Java)

This repository documents my learning journey with Selenium WebDriver using Java.
It focuses on setting up the automation environment and executing basic browser automation as a beginner transitioning from manual testing to automation testing.

## Tools & Technologies
- Java 11
- Selenium WebDriver (Selenium 4)
- Maven
- IntelliJ IDEA
- Google Chrome browser

## What is implemented so far
- Verified Java 11 setup using a simple Java program
- Configured Maven project and added Selenium WebDriver dependency
- Successfully launched and closed Chrome browser using Selenium WebDriver
- Automated opening of Google homepage

## Project Structure
- `src` – Java source code
- `pom.xml` – Maven configuration and dependencies
- `screenshots/` – Execution proof screenshots

## Notes
This repository represents hands-on learning and practice.
More Selenium features and test scenarios will be added incrementally as part of continuous learning.
